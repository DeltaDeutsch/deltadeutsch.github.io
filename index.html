Web Design Challenge: Weather Insights from Steve Deutsch!

Plotting Weather Data

The purpose of the project was to analyze how weather changes as you get closer to the equator. To accomplish this analysis I pulled the data from the OpenWeatherMap API to assemble a dataset of over 500 cities. I then used Matplotlib to plot various aspects of the weather such as temperature, cloudiness, wind speed, and humidity vs. latitude.
